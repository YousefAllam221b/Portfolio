    • The Scheduler

    The program creates possible full schedules by taking an excel sheet with only tutorial slots filled with no lectures. It also takes lists of all lecture slots avaliable for each course needed then tries possible ways to create a schedule with full lecture slots. It also have a few options to create a schedule with a specific features. When all options are counted for, the program either outputs an excel file with multiple sheets or it will tell if there were no possible outcomes with the options given by the user. It could be that the there were no possible ways witht the choosen features or it is not possible to create a schedule with the lecture slots options given. The program lacks the interface but I will be working on it as soon as I learn the needed skills. The program uses Google Calendar API
